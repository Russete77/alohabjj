# Templates de arte — AlohaBJJ / @bjjcomlucas

Assets estáticos do frame de Story/Reel (servidos em /templates/... pelo Next).

## Coloque aqui:
- `story-frame.png`   → o layout teal completo que você mandou (referência visual)
- `atleta.png`        → recorte do atleta (fundo TRANSPARENTE, gi Hayabusa) — canto inferior esq.
- `avatar.png`        → avatar @bjjcomlucas (círculo, topo direito)
- `logo.png`          → (opcional) logo AlohaBJJ

O template HTML/CSS (web/lib/templates/story-frame) referencia estes arquivos por
/templates/<nome>. Enquanto não estiverem aqui, o render usa placeholders.
